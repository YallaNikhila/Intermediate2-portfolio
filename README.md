# Personal Portfolio Website

## Project Overview

This project is a Responsive Personal Portfolio Website created using HTML5 and CSS3 as part of the Developers Arena Internship Week 2 Task.

The portfolio website contains:
- About Section
- Skills Section
- Contact Form
- Navigation Menu
- Footer

The main objective of this project is to learn CSS styling, responsive design, Flexbox layouts, hover effects, and webpage structure.

---

- HTML5
- CSS3
- Google Fonts
- Flexbox
- Media Queries

# CSS Concepts Used

## External CSS
An external CSS file named `style.css` was created and linked to the HTML file.

## CSS Selectors
Different CSS selectors were used:
- Element Selector
- Class Selector
- ID Selector

## Flexbox Layout
Flexbox was used to align content and create responsive layouts.

## Hover Effects
Hover effects were added on:
- Navigation links
- Skill boxes
- Buttons

## Responsive Design
Media queries were used to make the website responsive for mobile devices.
